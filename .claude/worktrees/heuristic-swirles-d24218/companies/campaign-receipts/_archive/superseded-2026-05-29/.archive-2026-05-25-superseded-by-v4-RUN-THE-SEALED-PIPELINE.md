**Audience:** founder (Alex) OR any agent invoking the `sealed-production-pipeline` skill.
**Promise:** every step is deterministic. Given the same inputs, you get the same outputs. Every gate is binary PASS/FAIL with explicit error messages. No magic, no shortcuts.

## The minimum inputs to start a piece

You need exactly 4 things. If any is missing, the pipeline STOPS at Stage 0 and asks:

1. **Topic brief** — markdown at `briefs/<YYYY-MM-DD>-<slug>-topic-brief.md`. 2-5 sentences. Example: "Promise #74: 4 presidents delayed Jerusalem for decades, then on May 14, 2018 the ribbon cut while 60 died at the Gaza border. Grade: KEPT."
2. **Slug** — kebab-case (e.g., `sealed-008-china-currency`). Becomes the folder name and Remotion episode key.
3. **Program** — `sealed` (145-promise audit, parchment thumbs, sealed2016.com CTA) OR `cr-new-news` (single-race finance receipt, navy thumbs, campaignreceipts.com/race/<slug> CTA).
4. **Mode** — `longform` (with VO) OR `short` (text + music + b-roll, NO VO per 2026-05-24 lock).

## End-to-end run — phases mapped to step numbers in the doctrine

### Phase 0 — preflight (founder, ~2 min)

```bash
SLUG=sealed-008-china-currency
PROGRAM=sealed
MODE=longform   # or "short"

# Create the brief
mkdir -p briefs research content/scripts eng/storyboards reports/episodes reports/council _review
cat > "briefs/$(date +%Y-%m-%d)-$SLUG-topic-brief.md" <<'EOF'
<!-- RECEIPTS: china, currency-manipulator, treasury-report, fec-id -->

Promise #N: candidate said he'd label China a currency manipulator on day one.
Treasury Department report under his admin: not labeled until August 2019,
de-labeled January 2020 as part of Phase One trade deal. Grade: PARTIAL.
EOF

# Scaffold Remotion episode if it doesn't exist (LF only; shorts re-use parent LF episode)
[ ! -d "../../remotion/src/episodes/$SLUG" ] && python3 scripts/pipeline/scaffold-remotion-episode.py --slug "$SLUG" --program "$PROGRAM"

# Invoke the skill — in Claude Code:
#   /sealed-production-pipeline --program $PROGRAM --mode $MODE
# (with the topic brief mentioned in your message)
```

The skill reads `SEALED-PRODUCTION-PIPELINE-v3.md` and runs the steps below.

### Phase 1 — Concept gate (Steps 1-6, ~10 min, ~$1 persona spend)

| Step | What happens | Hard fail? |
|------|--------------|-----------|
| 1 | Analytics tracker reads prior 72hr report → carry-over list informs the next piece | No (warning if no prior report) |
| 2 | Story investigator extracts tension + arc + reveal + stakes + 5-9 beats | Schema gate (5 H2 sections required) |
| 3 | Audience demand strategist returns GO/HOLD/KILL with hooks + CTR forecast | **KILL halts the pipeline** |
| 4 | Viral packaging lens scores hook strength, picks title direction | No (informs Stage 20) |
| 5 | Research persona surfaces primary sources for every claim | No |
| 6 | **Fact-check QC pre-script** — every claim must have a primary source | **HARD FAIL** — fix sourcing or abort |

**Pause point:** If Stage 3 returns KILL, the skill stops and surfaces the verdict. Founder either overrides (`--override-kill`) or accepts the abort.

## Resuming a partially-run pipeline

The skill is **idempotent per stage**. If a stage's OUTPUT artifact exists on disk and is fresh (mtime newer than the previous stage's output), the skill skips that stage and moves to the next.

Make recovery cheap:
- Bad story extraction? Delete `briefs/<date>-<slug>-story-extraction.md`, re-invoke skill — picks up at Stage 2.
- Bad script? Delete `content/scripts/<slug>-v5-locked.md`, re-invoke — picks up at Stage 7 (or whichever earlier vN is missing).
- Bad master? Delete `public/<mode>/<slug>.mp4`, re-invoke — picks up at Stage 24.

## Founder override paths

These exist but require explicit human commit:

| Override | Flag / file | Use case |
|----------|-------------|----------|
| KILL → GO | `--override-kill` at skill invoke + comment | Founder believes the topic has demand the strategist missed |
| Council <10/10 → ship | Founder edits the council report + adds `OVERRIDE: <rationale>` line | Time-sensitive piece; founder accepts the risk |
| TTS lint fail → ship anyway | Founder edits `--strict-lint` to `--lint` in the invocation | Almost never correct; logged loudly |
| No-VO gate fail → ship as-is | NOT ALLOWED. Hard rule per 2026-05-24 lock. Rebuild without VO. | n/a |

## Per-stage cost log

Every stage writes to `scripts/.external-costs.jsonl`:

```json
{"ts": "...", "issueId": "<slug>", "stage": N, "vendor": "anthropic/sonnet-4.6", "cost_usd": 0.12, "note": "story-extraction"}
```

Founder can audit per-piece spend with:
```bash
jq -r 'select(.issueId == "<slug>") | "\(.stage)\t\(.cost_usd)\t\(.vendor)\t\(.note)"' scripts/.external-costs.jsonl
```

Total-spend cap is enforced inside the skill — pipeline pauses if a piece exceeds $5 persona OR $25 render. Founder approves with `--budget-override <usd>`.

## Glossary

| Term | Meaning |
|------|---------|
| **Sealed-2016** | The 145-promise audit program. Parchment-aesthetic thumbs. Voice: "we already wrote the verdict; here's the receipt." |
| **CR new-news** | The race/finance receipt program (Bush MO-1, Massie KY-4, future). Navy thumbs with politician portraits. Voice: "$N moved into ONE district; here's the FEC trail." |
| **AY-pack** | TTS-safe spelling of AIPAC. Bare AIPAC fails Stage 13 lint. |
| **145 PROMISES card** | The standard CTA card at the end of SEALED shorts. Don't drop this. |
| **Episode-scoped Remotion** | Compositions live under `remotion/src/episodes/<slug>/` and render as `{slug}__{composition}` (2026-05-23 lock). Bare composition IDs are banned. |
| **youtubeBroll** | The episode-JSON block declaring a real-footage clip from YouTube. Wins over `fal-video` on the same beat per Doctrine line 5. |
| **CSPAN-burn-in** | The lower-third graphic on C-SPAN YouTube uploads. Drop via `cropTopPct: 0.75` per `eng/YOUTUBE-BROLL-AUDIT-SEALED-SHORTS.md`. |
| **Studio upload** | Manual MP4 drag-drop at studio.youtube.com — does NOT count against the project's `Video Uploads per day` cap. Mandatory per founder rule 2026-05-23. |
