# Citation Directory Submissions — CampaignReceipts

Tracking log for outbound submissions to citation archives, data catalogs,
and research directories. Goal: slow-compound brand authority + permanent
backlinks + Wikipedia ripple effect.

Each `*.md` in this folder = one directory. Status field is one of:

- **SUBMITTED** — sent; awaiting confirmation
- **ACCEPTED** — confirmed listed
- **PENDING-REVIEW** — under maintainer review
- **REJECTED** — declined (record reason)
- **TODO-HUMAN** — requires CAPTCHA / login / human action by founder

Last batch: 2026-05-19.
