## Hero concept (the "entering the Matrix" feel)

### Visual language

The hero animation captures the moment a user "enters" their hyperlocal
chat scope — visually echoing the iconic Matrix aesthetic but updated
for 2026:

- **Initial state:** dark interface, faint geometric patterns
  reminiscent of code rain (NOT literally the green Matrix code — too
  on-the-nose)
- **Avatar/user POV:** as user scrolls or interacts, lines of nearby
  conversations materialize from the periphery (subtle motion, NOT
  flashy)
- **Geographic anchor:** subtle radius indicator (5-mile circle) that
  pulses softly around the user's avatar
- **Chat snippets float** at varying distances ("0.3 mi away," "1.2 mi
  away," "4.8 mi away") — distance is part of the aesthetic
- **The 'reveal moment':** when user clicks "Enter," the radius
  expands and chat snippets resolve into readable conversations

### Color palette

- **Primary:** deep slate / charcoal (#0E1116 to #1A1F2B range)
- **Accent 1:** electric green (#22D87B) — "you are here" indicator,
  active state
- **Accent 2:** amber (#FFB940) — distance indicators, business
  channels
- **Text:** off-white (#E6E8EC), muted grey (#8B92A0) for distance
  labels
- **NOT** literal Matrix green-on-black (cliché); subtle homage instead

### Typography

- **Heading:** monospace mixed with sans-serif (e.g., JetBrains Mono
  for distance/data, Söhne or Inter for prose)
- **Body:** modern sans (Inter, IBM Plex Sans)
- **Voice texture:** anonymous-mysterious-but-friendly (NOT creepy)

## Below-utility: how it works (3 steps)

### Step 1: Anonymous to your neighbors
Random username. No real name. Your face never shows.

### Step 2: Verified to the platform
18+ verification + credit card on file. We know you're real (so the
trolls can't ruin it for everyone).

### Step 3: 5-mile radius by default
Free users see 5 miles. Paid premium ($5/mo) lets you jump into any
neighborhood across the city.

## What the landing page should NOT be

- Not Nextdoor's "wholesome neighborhood community" aesthetic
  (we're explicitly anti-Nextdoor)
- Not Yik Yak's "wild west anonymous" aesthetic (we have moderation)
- Not dating app aesthetic (different market, wrong implication)
- Not B2B SaaS clean-template (this is consumer)
- Not motivational guru aesthetic
- Not "trust us, it's safe!" overpromise

---

## Brand differentiation reinforced

The landing page reinforces the wedge:

1. **Anti-Nextdoor:** "anonymous to your neighbors" (Nextdoor uses
   real names)
2. **Anti-Yik Yak:** "verified to platform" (Yik Yak was pure
   anonymous → death spiral)
3. **Anti-dating-app:** "ask about dessert, not hookups" (utility
   scenarios make this clear)
4. **Anti-business-spam:** "verified businesses, proximity-gated"
   (not a Yelp clone)

## Engineering notes for Frontend Engineer

- Hero animation: CSS + Framer Motion, NOT WebGL/Three.js (overkill)
- Mock chat scenarios: static React components, no live data
- Mobile-first responsive (most visits will be mobile)
- Accessibility: keyboard nav, screen reader friendly, animations
  respect `prefers-reduced-motion`
- Performance: Lighthouse score 90+ on mobile
- No heavy assets: keep landing page <500KB initial load
