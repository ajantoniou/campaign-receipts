# SEALED 2016 Production Pipeline v3 — canonical doctrine

**Status:** LOCKED 2026-05-24. Supersedes `eng/PRODUCTION-PIPELINE-STEPS.md` (the in-flight v2 flowchart) for SEALED program work specifically. CR new-news pipeline (race/finance receipts) still uses the same skill in `cr-mode` — see appendix.
**Author:** Claude, after the 2026-05-21→24 rebuild cycle exposed every shortcut path (wrong cached b-roll, missing Remotion episodes, AIPAC pronunciation drift, parchment-only short thumbs, 145-promises card fatigue, shorts-with-VO cutoffs).
**Founder direction:** "simplify so an idiot AI agent can't take shortcuts. Each step is one action, one persona, one tool."
**Skill wrapper:** `.claude/skills/sealed-production-pipeline/SKILL.md` runs the steps end-to-end via Agent subagents.
**Sister doc:** `RUN-THE-SEALED-PIPELINE.md` — operations how-to.

## Doctrine in 8 lines

1. **One persona, one job, one tool.** No persona "gives advice" — each one PRODUCES one named artifact. Every step has a binary PASS/FAIL gate.
2. **Hard-fail gates between every stage.** Bad output blocks the next stage, period. Founder-override paths require explicit human intervention — never an AI shortcut.
3. **Receipts before rhetoric.** No factual claim ships without a primary-source cite (FEC filing, Federal Register, SEALED book page, certified election result, UN roll call). Hallucinated numbers kill the channel — same standard for both parties.
4. **Voice over LF; text + music + b-roll on shorts.** Long-forms carry the Betsy / Jessica VO. **Shorts use NO voice** — on-screen text cards + music + real footage only. Cutoffs on mobile feed kill retention; founder lock 2026-05-24.
5. **Real footage beats AI fills on hook/payoff beats.** YouTube-broll lane (`youtubeBroll` in episode JSON) wins over `fal-video` on the same beat. Generative only when no real footage exists for that moment.
6. **Cost discipline.** Each persona stage logged to `.external-costs.jsonl`. Soft cap $5/episode on persona spend, hard cap $25/episode on render spend, founder override required to exceed.
7. **9:16 vertical is a first-class Remotion target.** Every composition MUST detect `useVideoConfig().width/height` and scale layout when `height > width`. No 16:9-only compositions. Pattern: `const k = isVertical ? 2.2 : 1.0`, multiply font sizes + padding by `k`, set `maxWidth: isVertical ? width * 0.92 : <LF default>`. Horizontal compositions (timelines, money-flow diagrams) that don't translate need a `VerticalXxx` sister component. Founder lock 2026-05-24 after the 002-017 batch /watch QC found 7 shorts with floating-in-middle text. Reference fix: `remotion/src/compositions/SourceCard.tsx` (scale-multiplier) + `remotion/src/compositions/Timeline.tsx` (`VerticalTimeline` sister). New compositions FAIL Stage 22 if rendering at 1080×1920 produces >25% empty vertical space with no intentional design reason.
8. **Visual QC is a real gate, not a checklist.** Stage 25-26 must run an automated `qc-visual-master.py` (the `/watch` skill plus structured-output rubric) on every short and LF master. Findings that fail the gate: text cropped at frame edges, text card visible <1.2s, empty-frame moments >0.5s mid-animation, verdict stamp lands at wrong timestamp vs sync_specs, b-roll content mismatches attribution sidecar (the 2026-05-23 cache-id bug class), on-screen text spelling drifts from script text-cards, total runtime >60s for shorts. See `eng/QC-VISUAL-CHECKLIST.md`.

## Pronunciation + brand banlist (HARD-FAIL gates apply pre-TTS)

| Term | TTS form | Why |
|------|----------|-----|
| AIPAC | `AY-pack` | TTS reads "AI pac". Memory: `feedback_aipac_pronunciation.md` |
| AIPAC #N | `AY-pack #N` | Same. On-screen card "AIPAC" is fine; spoken must be hyphen-spelled |
| 145 promises | `One hundred forty-five promises` | Number-words handled per `brand/voice-writing.md` |
| sealed2016.com | `SEALED twenty sixteen dot com` | spoken form for CTA |

Any spoken script containing bare `AIPAC`, `AIPAC#`, `AIPC` fails Step 13 lint gate → block TTS spend.

## What this doctrine does NOT allow

- ❌ `videos.insert` from any agent (founder manual Studio upload only — preserves the ~6/day Video Uploads cap)
- ❌ Shorts with VO (Doctrine line 4 — hard fail)
- ❌ Bare `AIPAC` in spoken text (use `AY-pack`)
- ❌ Generic Remotion `bar2` for SEALED election shorts (use `cr-election-margin` per Remotion episode lock 2026-05-23)
- ❌ AI-generated faces of real politicians (b-roll only)
- ❌ Hallucinated FEC numbers, dates, or vote tallies — every figure traces to a primary source
- ❌ Loosening a hard-fail gate without explicit founder override (logged)
