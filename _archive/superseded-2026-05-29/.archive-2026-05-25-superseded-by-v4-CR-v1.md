**Status:** LOCKED 2026-05-24. Supersedes the narrative runbook in `eng/PRODUCTION-PIPELINE-RUNBOOK.md` (kept as background reading, but the table below is the source of truth for orchestration).
**Author:** Claude, after the 2026-05-22 → 2026-05-24 Bush + Massie LF rebuild cycle exposed every shortcut path (URANIUM chart leak, stale `public/longform/`, shorts shipped with mid-sentence cuts and letterboxed gutters, AIPAC mispronounced as "Aypack").
**Skill wrapper:** `companies/campaign-receipts/.claude/skills/cr-production-pipeline/SKILL.md` runs all steps end-to-end via Agent subagents.

---

## Doctrine in 7 lines

1. **One persona, one job, one artifact.** No persona "consults" — each one PRODUCES one named file on disk.
2. **Hard-fail gates between every stage.** Bad output blocks the next stage. No silent loosening.
3. **Verify before quoting.** Numbers in scripts (FEC totals, vote margins, percentages) must cite source URL + retrieval date. Hallucinated numbers in a campaign-finance video kill the channel.
4. **Single-narrator (Betsy / ElevenLabs Jessica).** Per-speaker voice settings locked in `personas/betsy-the-narrator.md`. Force-pronounce acronyms via `A.I.P.A.C.` style spelling in the script (not in on-screen text).
5. **Visuals serve the receipt.** Wikipedia photos + Remotion charts/text-cards first. Generative AI (fal Sora/Kling) only for atmosphere beats where no real photo exists. NEVER an AI-generated face of a real politician.
6. **Shorts are TEXT + B-ROLL + MUSIC, not VO clips.** Founder lock 2026-05-24: voice-clip shorts always fail one of {mid-sentence cut, letterbox gutter, on-screen text overflow}. Replaced with full-bleed b-roll + Remotion text cards + music bed.
7. **Cost discipline.** Per-episode persona spend ≤ $5 (soft). Per-episode render spend ≤ $20 (hard). Stripe-style logging to `scripts/.external-costs.jsonl` per stage.

## Shorts sub-pipeline (Step 35 expanded)

**Founder lock 2026-05-24 (HARD-FAIL DOCTRINE):** Shorts are **TEXT + B-ROLL + MUSIC, NOT VO CLIPS**. Cutting from the long-form VO always fails one of: (a) mid-sentence cut, (b) on-screen text overflow, (c) viewer can't follow without audio (40% of Shorts viewers watch muted). The default short format is full-bleed b-roll/photo + Remotion text cards + music bed. **No spoken VO.**

### How to add a tool to the matrix

When founder adopts a new vendor:
1. Define its UNIQUE strength (one sentence — what is it best in class at?)
2. Define what it LOSES to other tools
3. Assign the persona who owns prompt-craft for it
4. Slot it into a stage in the pipeline
5. Add a row to this table
6. Add a hard-fail rule to `visual-prompt-engineer.md` rejecting beats that try to use it for the wrong job

### How to add to this table

When a failure ships:
1. Add a row with date, symptom, root cause, the gate that should have caught it
2. If no existing gate would have caught it, add a new gate stage + new script
3. Update the relevant persona's `.md` file with a "DO NOT" clause mapping to this failure
4. The new gate must be hard-fail (exit ≠ 0) — not a warning

## How an agent runs this pipeline

Invoke `/cr-production-pipeline` and pass the topic brief. The skill:

1. Confirms 4 inputs: topic brief, slug (`cr-<race>-<politician>-<seat>`), program (`cr-new-news` or `sealed-2016`), and optional `<!-- CONCEPTS: ... -->` header
2. Reads this doc as source-of-truth stage list
3. Runs Stages 1 → 35 in order, spawning Agent subagents and Bash tool calls per stage
4. Verifies the OUTPUT artifact after each stage
5. On hard-fail gate trip: STOPS, surfaces violation, asks founder (override / fix / abort)
6. At Stage 30 (founder manual upload), pauses and asks for the video ID
7. At Stage 35.f (per-short manual upload), pauses per short
8. Logs persona + render spend to `scripts/.external-costs.jsonl` per stage
9. Idempotent: if a stage's OUTPUT exists with mtime newer than prior stage, skips it

## What's expressly forbidden (auto-fail)

- AI-generated face of a real politician (any tool — fal, Higgsfield, Hedra). Wikipedia photo or nothing.
- Looping a clip to fill a longer beat window
- Calling `videos.insert` to upload (founder manual only — see `feedback_founder_does_video_uploads.md`)
- Shipping a short with spoken VO (after 2026-05-24 lock) without founder override flag
- Shipping a master that hasn't passed 2x /watch on the checklist (Stages 25, 26)
- Skipping `text-card-sync-qc.py` (Stage 24) — that's the URANIUM-leak detector
- Re-uploading a slug that already has a live ID in `PUBLISHED-YOUTUBE.md` without `--replace-id` flag
- Writing `AIPAC` (not `A.I.P.A.C.`) in any line of VO copy
- Any persona producing an artifact outside its declared OUTPUT column

## Verification (the pipeline is "done" when)

- One end-to-end run produces a master that PASSES 2x /watch checklist on first try
- Wall time per LF episode ≤ 6 hours
- Total cost per LF + 3 shorts ≤ $20 (render + personas)
- An idiot AI agent cannot take a shortcut because every gate exits non-zero on violation
- Each persona has ONE job, no overlap
